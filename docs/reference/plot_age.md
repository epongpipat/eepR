# plot_age

plot_age

## Usage

``` r
plot_age(data, y_var, x_var = "age", y_lab = NULL, x_lab = "Age")
```

## Arguments

- data:

  input data.frame

- y_lab:

  y-axis label

- x_lab:

  x-axis label

- var_y:

  y-axis variable to plot

- var_x:

  x-axis variable to plot

## Value

ggplot figure of y-axis variable and age
