# plot_scree

plot_scree

## Usage

``` r
plot_scree(sv = NULL, eigs = NULL, p_perm = NULL, a_thr = 0.05)
```

## Arguments

- sv:

  singular values

- eigs:

  eigenvalues

- p_perm:

  permuted p values for each component

- a_thr:

  alpha threshold for significance of permuted p values (default: 0.05)
