# split_dseg

split_dseg

## Usage

``` r
split_dseg(in_dseg, in_lut, out_dir, overwrite = FALSE)
```

## Arguments

- in_dseg:

  path to discrete segmentation file

- in_lut:

  path to look up table .tsv file (bids format)

- out_dir:

  output directory for the individual segmentation files

- overwrite:

  flag to overwrite output files (default: 0)
